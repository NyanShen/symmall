## 项目结构
symmall/
├── pom.xml                # 父工程：统一依赖版本
├── symmall-common         # 公共模块：工具类、常量、枚举、异常、响应体
├── symmall-framework      # 框架模块：配置、拦截器、全局处理、权限核心
├── symmall-system         # 系统管理模块：用户、角色、菜单、字典、日志
├── symmall-file           # 文件模块：上传、下载、OSS
├── symmall-mall           # 商城模块：商品、订单、购物车、支付
├── symmall-miniapp        # 小程序接口模块：授权、小程序专用接口
└── symmall-admin          # 管理后台启动模块：后台API入口

## 每个模块结构
模块/
├── controller/        # 接口层（继承-控制器父类）
├── service/           # 业务层
│   ├── impl/          # 实现类
├── mapper/            # 数据访问层
├── entity/            # 数据库实体
├── vo/                # 前端返回视图
├── dto/               # 前端传入参数
└── constants/         # 模块常量

## symmall-common
symmall-common/
├── exception/          # 全局异常、业务异常
├── result/             # 统一返回结果 R.java / Result.java
├── utils/              # 工具类：Date、Http、MD5、JWT、Snowflake
├── constants/          # 全局常量
├── controller/         # 控制器父类
├── enums/              # 全局枚举
└── annotation/         # 自定义注解（日志注解、权限注解）
